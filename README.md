# Portfolio de Sergio Canales (Proyecto React)

Este es el código fuente del portfolio personal de Sergio Canales, un desarrollador Full Stack. El sitio web está diseñado para ser minimalista, moderno y totalmente responsivo, mostrando información sobre el desarrollador, sus proyectos y habilidades.

## Tecnologías Utilizadas

Este proyecto se ha desarrollado utilizando las siguientes tecnologías:

- HTML5
- CSS3
- JavaScript (ES6+)
- **React**
- **Vite**

## Cómo Ejecutar el Proyecto

Para ejecutar la aplicación en tu entorno local:

1.  **Instala las dependencias:**
    ```bash
    npm install
    ```
2.  **Inicia el servidor de desarrollo:**
    ```bash
    npm run dev
    ```
    Esto iniciará la aplicación en modo de desarrollo y podrás acceder a ella a través de tu navegador (generalmente en `http://localhost:5173/`).
