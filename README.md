# Portfolio de Sergio Canales

Este es el código fuente del portfolio personal de Sergio Canales, un desarrollador Full Stack. El sitio web está diseñado para ser minimalista, moderno y totalmente responsivo, mostrando información sobre el desarrollador, sus proyectos y habilidades.

## Estructura del Proyecto

El proyecto se compone de los siguientes archivos:

- `index.html`: La estructura principal del sitio web.
- `style.css`: Los estilos que dan vida al diseño del sitio.
- `script.js`: La lógica de JavaScript para la interactividad del sitio.
- `img/`: La carpeta que contiene las imágenes utilizadas en el proyecto.

## Prácticas y Heurísticas de Diseño

Este proyecto se ha desarrollado siguiendo una serie de buenas prácticas y heurísticas de diseño para garantizar un producto de alta calidad, accesible y mantenible.

### Accesibilidad (a11y)

La accesibilidad ha sido una prioridad en el desarrollo de este proyecto. Se han implementado las siguientes características para garantizar que el sitio sea utilizable por la mayor cantidad de personas posible:

- **Contraste de color:** Se ha prestado atención al contraste de color para garantizar que el texto sea legible para personas con discapacidad visual.
- **Texto alternativo en imágenes:** Todas las imágenes tienen un atributo `alt` que describe su contenido, lo que permite a los lectores de pantalla transmitir esta información a los usuarios.
- **Navegación por teclado:** El sitio es totalmente navegable utilizando solo el teclado.
- **ARIA (Accessible Rich Internet Applications):** Se han utilizado atributos ARIA para mejorar la accesibilidad de los componentes dinámicos, como el menú de navegación móvil.
- **Semántica HTML:** Se ha utilizado HTML semántico para estructurar el contenido de forma lógica y comprensible para los lectores de pantalla.

### Diseño Responsivo

El diseño del sitio es totalmente responsivo, lo que significa que se adapta a diferentes tamaños de pantalla, desde teléfonos móviles hasta ordenadores de escritorio. Se ha utilizado un enfoque "mobile-first", diseñando primero para pantallas pequeñas y luego añadiendo estilos para pantallas más grandes. Para garantizar que las imágenes mantengan su proporción y no se distorsionen, se utiliza la propiedad `object-fit: cover` en las tarjetas de los proyectos.

### Minimalismo

El diseño del sitio es minimalista, centrándose en el contenido y eliminando cualquier elemento innecesario. Esto no solo mejora la estética del sitio, sino que también mejora la experiencia del usuario al facilitar la búsqueda de información.

### Rendimiento

Se ha prestado atención al rendimiento del sitio para garantizar que se cargue rápidamente. Se han optimizado las imágenes y se ha minimizado el uso de JavaScript. Además, se utiliza el atributo `loading="lazy"` en las imágenes para diferir la carga de las que no están en la vista inicial, mejorando así el tiempo de carga percibido.

## Tecnologías Utilizadas

Este proyecto se ha desarrollado utilizando las siguientes tecnologías:

- HTML5
- CSS3
- JavaScript (ES6+)
